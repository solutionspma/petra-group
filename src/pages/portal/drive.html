My Drive

Private cloud storage for TPG. With Supabase configured, you sign in with your work account and files are private per user. Until then, this page runs in demo mode — uploads stay in this browser only (localStorage), no server.

Demo mode — Try upload, download, and delete. Data never leaves this device. Add SUPABASE_URL + SUPABASE_ANON_KEY on Netlify and redeploy for real cloud storage. Append ?demo=1 to this URL to force demo even if Supabase is set.
Drive not configured on this build.

Add environment variables SUPABASE_URL and SUPABASE_ANON_KEY in Netlify (or your CI), then redeploy. Run the SQL in supabase/drive-setup.sql in the Supabase SQL editor. See comments in that file for creating users and the private bucket.
